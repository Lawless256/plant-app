Drop plant photos in this folder.

Then in src/data/quiz.js, reference them like this:
  photo: './photos/wandering-dude.jpg',

Tips:
- Use lowercase filenames with hyphens, no spaces (good: pink-princess.jpg)
- JPG or PNG both work
- Photos under 800px wide load faster — you can resize before saving
  (or use any free tool like https://tinypng.com to compress)
